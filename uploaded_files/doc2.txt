Java es un lenguaje de programación orientado a objetos, concurrente y multiplataforma, que fue diseñado por James Gosling en 1991. Java se ejecuta sobre una máquina virtual (JVM), lo que le permite ser compatible con diferentes sistemas operativos y arquitecturas. Java es uno de los lenguajes más populares y utilizados en el mundo, especialmente para el desarrollo de aplicaciones web, móviles y de escritorio . Java se caracteriza por ser simple, robusto, seguro, dinámico, portable y eficiente.
